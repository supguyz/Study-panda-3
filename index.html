🤔 How sharing works (no backend!)

  • We keep a tiny database in your browser (localStorage).
  • Use links so everyone can access the file.
    • Google Drive → Share → Anyone with the link → Viewer.
  • File uploads create a temporary link that works only on this device.
  • Want multi-user contributions? Use a Google Form to collect titles/links → publish the sheet → paste links here. Easy! ✨
